The ``add_integer`` function
=============================

Add two numbers, casting floats to integers first.

Basic usage
-----------

Import the function:

    >>> add_integer = __import__('0-add_integer').add_integer

Add two integers:

    >>> add_integer(1, 2)
    3

Add with a negative:

    >>> add_integer(100, -2)
    98

Use the default value for b:

    >>> add_integer(2)
    100

Cast floats to integers before adding:

    >>> add_integer(100.3, -2)
    98

    >>> add_integer(2.9, 3.9)
    5
