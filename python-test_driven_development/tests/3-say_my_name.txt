The ``say_my_name`` function
==============================

Print My name is <first name> <last name>.

Basic usage
-----------

Import the function:

    >>> say_my_name = __import__('3-say_my_name').say_my_name

Print a full name:

    >>> say_my_name("John", "Smith")
    My name is John Smith

    >>> say_my_name("Walter", "White")
    My name is Walter White

Use the default value for last_name:

    >>> say_my_name("Bob")
    My name is Bob 

The function returns None:

    >>> say_my_name("John", "Smith") is None
    My name is John Smith
    True

Error handling
--------------

Raise TypeError when first_name is not a string:

    >>> say_my_name(12, "White")
    Traceback (most recent call last):
    TypeError: first_name must be a string

Raise TypeError when last_name is not a string:

    >>> say_my_name("John", 98)
    Traceback (most recent call last):
    TypeError: last_name must be a string

Raise TypeError when first_name is None:

    >>> say_my_name(None)
    Traceback (most recent call last):
    TypeError: first_name must be a string

Raise TypeError when no arguments are given:

    >>> say_my_name()
